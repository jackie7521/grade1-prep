# 🎒 小一銜接 暑假 8 週快樂學習

一套專為「即將上小學一年級」的孩子設計的免費銜接教材，可在瀏覽器線上互動學習，也可列印成紙本練習本。

> ❤️ **設計理念：**不超前學習，而是讓孩子「習慣坐下來學」、「熟悉小學節奏」、「建立學習自信」，快樂迎接開學日。

---

## 🌐 線上版本（GitHub Pages）

把 repo 設為 Public 並開啟 GitHub Pages 後，可直接打開：

```
https://<你的帳號>.github.io/<repo 名稱>/
```

首頁就是互動教材；其他兩個檔案的網址：

- 家長指南：`/parent-guide.html`
- 列印練習本：`/workbook.html`

---

## 📂 檔案說明

| 檔案 | 用途 | 使用方式 |
|------|------|---------|
| `index.html` | **🎮 線上互動教材** | 用電腦或平板瀏覽器打開，孩子可以點擊互動、聽發音、答題拿星星 |
| `parent-guide.html` | **📘 家長指南** | 完整 8 週課程地圖、學習目標、家長陪伴 5 原則、進度檢核表 |
| `workbook.html` | **📝 可列印練習本** | 用瀏覽器打開後按 `Ctrl + P` (Mac: `Cmd + P`) 列印或存成 PDF |

## 🎯 建議學習方式

每天 **30-40 分鐘**，分 2 次完成：

- 🌅 **早上 20 分鐘**：國語 10 分 + 數學 10 分（精神好時學動腦的科目）
- 🌇 **下午 15 分鐘**：英語 8 分 + 生活 7 分（放鬆性內容）
- 📅 每週 5 天學習，週末休息或玩延伸活動

---

## 🚀 如何架設到 GitHub Pages

1. 把 repo 設為 **Public**（私人 repo 用 GitHub Pages 需付費版）
2. 進到 repo 的 **Settings → Pages**
3. **Source** 選 `main` 分支、`/ (root)` 資料夾
4. 按 **Save**，等 1-2 分鐘
5. 頁面上方會出現網址，點開就是 `index.html`

## 🛠️ 技術說明

- 純 HTML + CSS + JavaScript，**無任何後端、無外部相依**
- 線上版用瀏覽器內建 [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) 朗讀（需網路或系統支援中文 / 英文語音）
- 練習本用 `@media print` CSS，列印時自動分頁、隱藏按鈕
- 字型使用系統內建中文字型，無需額外載入
